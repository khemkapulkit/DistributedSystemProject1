Instructions to Execute the code

1.  Copy the AOS folder to the home folder of your linux machine on UTD Network.
	Alternatively, you could also copy the contents of Project1 folder to $HOME/AOS/Project1

2. Edit the 'config.txt' file placed in the Project1 folder as your requirement.

3. Copy the 'cleanup.sh' and 'launcher.sh' to your machine(Pref Home folder).

4. Execute 'launcher.sh'. 

5. Wait till all the nodes have recieved all their tokens.

6. Now press ctr-C to stop.

7. Execute 'cleanup.sh'.
